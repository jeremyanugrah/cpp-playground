# Hello World (C++)

Basic C++ test project using `std::vector` and range-based for loops.

## Build & Run

```bash
g++ main.cpp -o app && ./app
```

Or in Neovim: **Space + r**

## Environment

- **Compiler:** GCC/G++ via `build-essential`
- **OS:** ChromeOS + Debian 12 (Bookworm) Container
- **Editor:** Neovim (v0.7.2)

See the root [README](../../README.md) for full setup details and Neovim configuration.
